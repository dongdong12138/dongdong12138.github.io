# 样式

## Less & Sass

```shell title="安装"
npm install less -D

npm install sass -D
```

```html title="使用"

</style>

<style lang="scss">
 
</style>
```

## scoped

实现组件样式的私有化。

## BEM 架构

BEM：即 block（块层）、element（元素层）、modify（修饰符层）。

ElementUI 用的也是这种架构。

```css title="BEM 命名约定"
.block {}
 
.block__element {}
 
.block--modifier {}

.block__element--modifier {}
```
